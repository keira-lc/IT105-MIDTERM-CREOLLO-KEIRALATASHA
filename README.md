## Ordering and Inventory Management System

### Project Overview:
This project is an Ordering and Inventory Management System built using MySQL. It manages products, customers, orders, and order details. The system demonstrates database design, normalization, SQL operations, and indexing.
__________________________________________________________________________________________________________
### Database Structure:

 Tables:

1. Products – stores product details and stock
2. Customers – stores customer information
3. Orders – stores order transactions
4. OrderDetails – stores products per order
__________________________________________________________________________________________________________
### Features Implemented:
Database creation (MySQL)
Table relationships (foreign keys)
Data insertion (10+ records per table)
SQL queries (SELECT, INSERT, UPDATE, DELETE, JOIN, SUBQUERY)
Normalization (UNF → 3NF)
Indexing for performance improvement
GitHub version control (multiple commits)
