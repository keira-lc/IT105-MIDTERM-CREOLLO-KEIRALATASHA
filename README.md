Ordering and Inventory Management System

Project Description: 
This project is an Ordering and Inventory Management System designed to manage products, customers, orders, and stock levels. It helps track product availability, customer purchases, and order transactions efficiently.

Tables Description:
1. Products
Stores product details and stock quantity
2. Customers
Stores customer information
3. Orders
Stores order transactions
4. OrderDetails
Stores products included in each order

Features Implemented:
- Database creation using MySQL
- Table relationships using foreign keys
- Data insertion (10+ records each)
- SQL operations (CRUD, JOIN, Subquery)
- Normalization (UNF → 3NF)
- Indexing for performance optimization
